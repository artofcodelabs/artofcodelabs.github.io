 EkoSkaner is an offline-ready web app for ticket validation, integrated with Ekobilet via API.
Built with barcode scanner support, browser-side sync using Service Workers, and full offline mode.
EkoSkaner has been using the Loco framework since day one.

 Created solely by Zbig in the first 3 months of 2016.

#ruby #rails #coffeescript #loco #serviceworker

 Ekobilet (pol. bilet = ticket) is a ticket sales platform supporting both e-tickets and POS-printed tickets.
Used by event organizers to manage sales and distribution.

 Developed and maintained by Zbig between 2010 and 2017.
