About

Art of Code is a boutique software studio founded and led by Zbigniew Humeniuk — a seasoned Software Producer, hands-on Engineering Manager, and full-stack developer with a focus on FinTech and web applications.
Since 2008, Zbig has designed and built modern, scalable systems using Ruby on Rails, React, webpack, HTML5 and tailwindcss. He’s the creator of Loco, an open-source framework for building reactive Rails applications with structured frontend code.
Beyond development, Zbig leads engineering teams, architects solutions, fosters high-quality coding practices and advocates pragmatic programming.
